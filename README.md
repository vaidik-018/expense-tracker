# EXPENSE TRACKER

A full stack expense tracker web app built using:
  -flask(backend)
  -html, css, java script (frontend)
  -SQLite (database)

#FEATURE
-add expense
-view summary
-delete expense
-monthly analytics chart

#RUN LOCALLY
1. install dependecies
2. rum app.py
3. open http://127.0.0.1:5000
